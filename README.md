# vagrant-ci-jenkins
Vagrant and Ansible setup to spin up a VM with Jenkins up and running

## Pre-requisites

- virtualbox or another provider
- vagrant
- ansible
- vagrant box Ubuntu 14.04 - vagrant box add ubuntu/trusty64

## Usage

### Start new VM

copy Vagrantfile
vagrant up

### Update running VM

vagrant provision
